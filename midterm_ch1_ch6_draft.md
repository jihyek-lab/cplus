# C++ 중간고사 초안

범위: Chapter 1 ~ Chapter 6  
시험시간: 75분  
형태: 폐쇄형, 이해 중심 평가

## 1. 출제 방향

이 시험은 단순 암기보다 다음 능력을 평가하는 것을 목표로 한다.

- 기본 문법과 개념을 정확히 이해하는가
- 코드를 읽고 실행 결과나 오류 원인을 설명할 수 있는가
- 짧은 코드 구현을 통해 핵심 개념을 적용할 수 있는가

## 2. 권장 구성

총점 100점 기준:

- 이론/개념 설명: 25점
- 코드 해석 및 오류 분석: 45점
- 짧은 구현: 30점

문항 수 예시:

- 이론 2문항
- 코드 해석 3문항
- 구현 2문항

총 7문항 내외

## 3. 시간 배분 권장

- 이론: 15분
- 코드 해석: 35분
- 구현: 25분

75분 시험에서 이 정도 구성이면, 이해한 학생이 충분히 설명할 시간을 확보하면서도 시험이 지나치게 길어지지 않는다.

## 4. 문항 난이도 기준

- 쉬운 문제: 기본 개념 확인
- 중간 문제: 코드 해석, 함수 호출 결정, 출력 결과 예측
- 어려운 문제: 컴파일 오류 분석, 짧은 구현, 설계 의도 설명

권장 비율:

- 쉬움 30%
- 중간 50%
- 어려움 20%

## 6. 권장 문제 유형

### 유형 A. 개념 설명형

예:

- 값 전달과 참조 전달의 차이를 설명하시오.
- static 멤버 변수와 일반 멤버 변수의 차이를 설명하시오.
- 리턴 타입만 다른 함수 오버로딩이 불가능한 이유를 설명하시오.

의도:

- 정의 암기가 아니라 차이와 이유를 설명하게 한다.

### 유형 B. 코드 해석형

예:

- 다음 코드의 출력 결과를 쓰시오.
- 다음 호출에서 어떤 함수가 선택되는지 쓰시오.
- 생성자와 소멸자의 호출 순서를 쓰시오.

의도:

- 학생이 문법을 실제 실행 흐름과 연결해 이해하는지 본다.

### 유형 C. 오류 분석형

예:

- 다음 코드가 컴파일되지 않는 이유를 쓰시오.
- 모호성 에러가 발생하는 이유를 설명하시오.
- 코드를 어떻게 수정하면 되는지 쓰시오.

의도:

- "왜 안 되는가"를 설명할 수 있는지 평가한다.

### 유형 D. 짧은 구현형

예:

- 디폴트 매개변수를 갖는 함수를 작성하시오.
- 객체 수를 세는 static 멤버 변수를 포함한 클래스를 작성하시오.
- 기본 크기를 가지는 생성자를 작성하시오.

의도:

- 긴 프로그램 작성보다 핵심 개념을 정확히 코드로 옮기는 능력을 본다.
