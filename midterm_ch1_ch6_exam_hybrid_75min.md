# C++ 중간고사 하이브리드 운영안

- 범위: Chapter 1 ~ Chapter 6
- 시험시간: 75분
- 총점: 100점
- 구성: 객관식 / 짧은 판단 + 코드 작성 / 완성 + 코드 분석 / 오류 수정 + 출력 예측 / 심화 분석

## 유의사항

- 객관식은 정답 번호만 쓰시오.
- 짧은 분석 문제는 문제에 제시된 형식대로 간단히 쓰시오.
- 빈칸 문제는 빈칸에 들어갈 코드 또는 단어만 쓰시오.
- 코드 작성 문제는 번호가 붙은 칸에 맞는 코드 한 줄 또는 표현을 쓰시오.
- 코드 작성 문제의 각 칸 배점은 문항에 표시된 점수를 따른다.
- 코드 분석 / 오류 수정형은 원인과 수정 내용을 구분해서 쓰시오.
- 주관식은 문제에 제시된 항목 순서대로 짧게 답하시오.
- 특별한 언급이 없으면 `#include <iostream>`, `#include <string>`, `using namespace std;`는 이미 있다고 가정한다.
