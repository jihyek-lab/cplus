## 10번. static 멤버와 짧은 코드 작성 (10점)

다음 `Counter` 클래스의 빈칸을 채우고, 아래 물음에 답하시오.

```cpp
class Counter {
private:
    int value;
    static int count;
public:
    Counter(int v = 0) : value(v) { ________; }
    ~Counter() { ________; }
    int getValue() const { return value; }
    static int getCount() { return count; }
};

int Counter::count = 0;
```

물음:
- static 멤버 변수 `count`는 어떤 의미를 가지는가?
- 왜 static 멤버 함수에서는 non-static 멤버 변수에 직접 접근할 수 없는가?

답안 가이드:
- 1) 첫 번째 빈칸 채우기
- 2) 두 번째 빈칸 채우기
- 3) `count`가 객체별 값인지, 클래스 공유 값인지 쓰기
- 4) static 함수와 `this`의 관계를 쓰기
