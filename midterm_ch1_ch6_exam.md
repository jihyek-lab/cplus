# C++ 중간고사 포괄형 시험지

- 범위: Chapter 1 ~ Chapter 6
- 목적: 범위 전체의 핵심 개념을 고르게 커버하는 출제안
- 비고: 이 버전은 "시간 제약을 우선하지 않고, 중요한 내용을 최대한 포함"하는 포괄형 시험지다.

## 유의사항

- 답은 결과만 쓰지 말고 이유를 함께 쓰시오.
- 구현형 문항은 `#include <iostream>`, `#include <string>`, `using namespace std;`가 이미 있다고 가정해도 된다.
- 특별한 언급이 없으면 `main()`은 생략 가능하다.
- 서술형은 핵심 개념이 분명하면 표현은 달라도 정답으로 인정한다.

### 23번. static 멤버 변수 해석 (10점)

다음 코드의 출력 결과를 쓰시오.

```cpp
class Person {
public:
    int money;
    static int sharedMoney;
};

int Person::sharedMoney = 10;

int main() {
    Person a, b;
    a.money = 100;
    b.money = 200;
    a.sharedMoney = 300;
    cout << a.money << ' ' << b.money << '\n';
    cout << a.sharedMoney << ' ' << b.sharedMoney << '\n';
}
```

또한 `money`와 `sharedMoney`의 차이를 설명하시오.
