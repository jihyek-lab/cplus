# C++ Midterm Exam Web Server

현재 폴더에는 `midterm_ch1_ch6_exam_student.md`를 기준으로 만든 로컬 시험 서버가 들어 있습니다.

## 포함 기능

- 학생용 웹 시험지
- 시험 제출 후 자동 채점
- 제출 결과 JSON 저장
- 관리자용 제출 목록 페이지
- 제출별 상세 점수 확인
- 전체 결과 JSON 내보내기

## 실행 방법

```bash
cd /Users/narim1pro/Documents/workspace/cplus/exam_web
python3 server.py
```

실행 후 접속:

- 시험지: `http://127.0.0.1:8000/`
- 관리자 목록: `http://127.0.0.1:8000/admin`
- 결과 JSON: `http://127.0.0.1:8000/admin/export`

## 제출 데이터 저장 위치

- 제출 파일 폴더: `data/submissions/`
- 각 제출은 JSON 파일 1개로 저장됩니다.

## 자동 채점 기준 메모

서버는 현재 학생용 시험지의 문항 점수 합계인 `100점` 기준으로 채점합니다.

기존 루브릭 문서에는 일부 문항에서 세부 배점 합이 문항 총점과 어긋나는 부분이 있었기 때문에, 서버에서는 아래처럼 일관되게 정리해서 채점합니다.

- 17번: `sum(3)` 4점, 나머지 두 호출 각 1점
- 18번: 오류 원인 2점, 수정 방법 2점
- 19번: 출력 결과 4점
- 22번: 각 코드 줄 2점씩
